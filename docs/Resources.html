Course Resources

Andreas Handel

2019-07-31

All materials described below are (should be) freely available online. If you can’t get to them, let me know.

General help

  • Google/Stackoverflow will be your best friend. Most of the time, someone had the same problem/question you do and someone else answered it. The only tricky thing is finding that post. Even after years of doing this, I probably google how to do something in R every day :)
  • R for Data Science community - an online community that tries to provide newcomer friendly help.

Github Resources

  • A great resource to learn Git/Github with R is happygitwitR by Jenny Bryant.
  • I recommend using a graphical client to do (almost) all your Github work (push, pull, etc.). I use GitKraken which has a fully functional free version. Jenny has a few other recommendations in her book.
  • The course materials website (this one) is a public Github repository. You can see all the files that make up this website here, the website itself is: https://andreashandel.github.io/MADAcourse/
  • For classwork, projects and other materials that we might not want to share publicly, I set up a Github organization called epid8060fall2019, you can find it here. Note that you need to be a group member.

Books

  • Introduction to Data Science (IDS) is the book we’ll use the most in this course. You will be assigned differet chapters to read and work through.
  • R for Data Science teaches some coding integrated with the material covered in the book. Very good and worth checking out often.
  • An Introduction to Statistical Learning - good introductory book focusing more on the data analysis side of things (as opposed to getting/wrangling/exploring data).
  • Hands-on Machine Learning with R. Good discussion of various analysis methods/approaches.
  • R Programming for Data Science - you can get it for free.

Online courses

  • Cloud Based Data Science - a nice online course covering many of the topics we cover at a somewhat more basic level. You can decide what to pay for it, including getting it for free. That course used to be called Chromebook Data Science and seems to be now updated and rebranded as Cloud Based Data Science.
  • “Data Science Specialization” on Coursera

Hands-on/interactive resources

  • Online Tutorials (primers) by Rstudio - we’ll do several of those in the class.
  • Swirl is a package that teaches R inside R. Especially beginner students have found it to be a good start since it provides very encouraging feedback. The downside is that all code writing happens interactively in the console, which is not the way one should write real code. It’s still worth checking out if you want to get some more direct, hands-on R practice
  • I used to recommend and use Datacamp, an online platform that has nice interactive courses teaching R and Data Analysis (and other topics). Unfortunately, the company recently dealt rather poorly with a case of sexual harassment and I therefore currently won’t use or recommend them anymore. That said, their courses are still good. Most of the material is not free and requires a subscription.
